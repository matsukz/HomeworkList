# 宿題を管理するやーつ

## スペシャルサンクス
* クリエイターとして活躍してる友だちくん
  * 無断でアイデアをパクってしまったことを謝罪します
  * アイデア料としてなんか奢るので許して
## 動作する環境
* Microsoft Excel
  * マクロ有効ブックが動作する環境
    * モバイル版やブラウザ版では動作しません
